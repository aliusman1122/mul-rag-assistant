## 📌 Project Overview

This is a **production-ready RAG (Retrieval-Augmented Generation) system** designed as an AI Information Assistant for **Minhaj University Lahore (MUL)**. It answers student queries about admissions, fee structures, programs, and more using AI + university documents.

### 🎯 Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI-Powered Q&A** | LLM-based answers using Groq (LLaMA 3) |
| 🔍 **Semantic Search** | ChromaDB vector database for intelligent search |
| 📄 **PDF Processing** | Automatic PDF ingestion and chunking |
| 🌐 **Web Scraping** | Auto-scrapes university website |
| 💬 **Chat History** | Persistent conversation tracking |
| 📊 **LangSmith Tracing** | Full observability and debugging |
| 🎨 **Professional UI** | Clean Streamlit interface |
| ♻️ **Reusable Template** | Clone and customize for any organization |

## 🚀 Installation & Setup

### Prerequisites
- Python 3.9+
- pip
- Git

### Step 1: Clone Repository
```bash
git clone https://github.com/YOUR_USERNAME/mul-rag-assistant.git
cd mul-rag-assistant
```

### Step 2: Create Virtual Environment
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Configure Environment
```bash
cp .env.example .env
```

Edit `.env` file and add your API keys:
```env
GROQ_API_KEY=your_groq_api_key_here    # Get free from console.groq.com
LLM_PROVIDER=groq
```

### Step 5: Add University Data
```bash
# Option A: Put PDFs in data/pdfs/ folder
cp your_university.pdf data/pdfs/

# Option B: Scrape university website (update SCRAPE_URLS in .env first)
python scripts/scrape.py
```

### Step 6: Ingest Data
```bash
python scripts/ingest.py
```

### Step 7: Start Application
```bash
# Start everything at once
python run.py

# OR separately:
uvicorn api.main:app --reload --port 8000
streamlit run ui/streamlit_app.py --server.port 8501
```

### Step 8: Access
- 🌐 **Chatbot UI:** http://localhost:8501
- 📚 **API Docs:** http://localhost:8000/docs

## ♻️ Reusable Template Guide

This project is designed as a **reusable RAG template**. To use it for a new organization:

### What to Change (Only 4 Things!)

| File | What to Change |
|------|----------------|
| `config/branding.py` | University name, colors, contact info |
| `config/settings.py` | `SCRAPE_URLS` list |
| `prompts/system_prompt.txt` | AI behavior and persona |
| `data/pdfs/` | New organization's documents |

### What Stays the Same
- All RAG components (`rag/`)
- All API routes (`api/`)
- All services (`services/`)
- UI framework (`ui/`)
- All scripts (`scripts/`)

### New University Checklist
```
✅ Clone this repository
✅ Update config/branding.py
✅ Update config/settings.py (SCRAPE_URLS)
✅ Update prompts/system_prompt.txt
✅ Add PDFs to data/pdfs/
✅ Run: python scripts/scrape.py
✅ Run: python scripts/ingest.py
✅ Run: python run.py
```

## 📊 LangSmith Tracing (Optional)

To enable AI tracing and debugging:

1. Create account at [smith.langchain.com](https://smith.langchain.com)
2. Get API key
3. Add to `.env`:
```env
LANGSMITH_API_KEY=your_key
LANGCHAIN_TRACING_V2=true
LANGCHAIN_PROJECT=MUL-RAG-Assistant
```
